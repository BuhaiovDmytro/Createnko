# Createnko Frontend

React-based frontend for Createnko video script generator.

## Development

```bash
npm install
npm start
```

## Build

```bash
npm run build
```

## Environment Variables

Create a `.env` file:

```env
REACT_APP_API_URL=http://localhost:8000
```

## Tech Stack

- React 18
- Tailwind CSS
- Framer Motion
- Axios
